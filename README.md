# DummyDump

Simple Database Backup Tool

## Getting Started

### Requirements

* Go >= 1.16.x

### Export

* How to run the program

```
go run ./cmd export <User Name> <Database Name>
```
## Import

* How to run the program

```
go run ./cmd import <User Name> <File Path> 
```
